# JSON Templates

⚠️ **IMPORTANT: These are EXAMPLE templates only**

All files in this directory contain **placeholder data** for demonstration and testing purposes:

- `applicant_profile.json` - Example profile using "Max Mustermann" (German placeholder name equivalent to "John Doe")
- `job_posting_example_ursapharm.json` - Example job posting
- `job_posting_template.json` - Template structure for job postings
- `example_history_with_tool_calls_and_tool_responses.json` - Example conversation history

**Do NOT use real personal data in these templates.**

When using the application:
1. Copy these templates as needed
2. Replace all placeholder data with your actual information
3. Keep your personal data files in ignored directories (e.g., `AppData/`)
4. Never commit files containing real personal information
